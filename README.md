# GroupOne Q&A Site

A web application for sharing and managing technical questions and answers.

## Prerequisites

- Node.js (v16+ recommended)
- npm or yarn
- A browser to run the frontend
- login page is working 

## Installation

1. Clone the repository:
```bash
git clone https://github.com/Luke-Manyamazi/GroupOne-QandA-Site.git
cd GroupOne-QandA-Site
```

## Install Dependencies
```bash
npm install
# or
yarn install
```

## Running the App
```bash
npm start
# or
yarn start
```
